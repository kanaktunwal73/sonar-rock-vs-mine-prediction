# sonar-rock-vs-mine-prediction
Machine Learning classification project using sonar data , Logistic Regression , Numpy , Pandas and Scikit-learn to classify underwater objects as Rock or Mine
# technologies used 
- python
- Numpy
- Pandas
- Scikit-learn
- LogisticRegression
# Features 
- Data Loading and Processing
- Data Analysis
- Train-Test Split
- Model Training
- Accuracy Evalution
- Rock vs Mine Prediction System
# Dataset 
  The project uses sonar signal data containing 60 numerical features and one target label :
- R = Rock
- M = Mine
# Result 
  The model predicts whether an underwater object detected by sonar signals is a Rock or a Mine
